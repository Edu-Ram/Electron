# Electron SQLite CRUD

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Compilar ejecutable

**Windows**

```bash
npm run dist:win
```

**Mac**

```bash
npm run dist:mac
```

**Linux**

```bash
npm run dist:linux
```

El ejecutable se genera en la carpeta `dist/`.
